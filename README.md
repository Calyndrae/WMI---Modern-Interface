##  Installation Guide

### 1. Prerequisite Extensions
Before running the installer, make sure you have these two extensions installed in your browser (Chrome or Edge recommended):
* [Tampermonkey](https://www.tampermonkey.net/) (The engine that runs the scripts)
* [Stylebot](https://stylebot.dev/) (The engine that runs the visual theme)

### 2. Run the Installer
Choose the installer that matches your computer:

#### üñ•Ô∏è Windows Users
1. Download `WMI_Installer.vbs`.
2. Double-click to run.
3. Follow the pop-up prompts. It will open each script in your browser; click **Install** or **Update** on each tab, then return to the installer to continue.

#### üçé macOS Users
1. Download `WMI_Mac_Installer.command`.
2. Open **Terminal** and type `chmod +x ` (with a space at the end).
3. Drag the `.command` file into the Terminal window and press **Enter**.
4. Double-click the file to run the installation sequence.
